Shell Commands

Fourteen commands covering navigation, developer info, GitHub data, and system display. Tab completion and fuzzy suggestions included.

💻

neofetch

System info display with ASCII art logo, Catppuccin color blocks, and live GitHub statistics.

visitor@rfs:~ $ neofetch
📁

ls / cd / cat

Navigate a virtual Unix filesystem. Grid and long format listing, directory traversal with ~ and - shortcuts.

visitor@rfs:~ $ ls -la projects/
🛠

skills

Technical skills with color-coded progress bars. Per-language Catppuccin colors and proficiency levels.

visitor@rfs:~ $ skills
📦

repos

GitHub repositories table sorted by stars. Languages, descriptions, and links from daily-updated data.

visitor@rfs:~ $ repos
📄

resume

Full resume with experience, education, and certifications. Rendered with box drawing and ANSI formatting.

visitor@rfs:~ $ resume
💬

contact

Contact information in a styled box. Email, social links, and profiles with clickable URLs.

visitor@rfs:~ $ contact

Features

A full terminal experience in the browser. Every interaction feels like a real shell session.

xterm.js Terminal

Full VT100/VT220 emulation with cursor movement, line editing, scrollback buffer, and 24-bit ANSI color rendering.

Virtual Filesystem

Navigable directory tree with ls, cd, cat, pwd. Grid and long format. Home, parent, and previous directory shortcuts.

Tab Completion

Context-aware completion for commands and file paths. Common prefix expansion. Cycles through candidates on repeated presses.

Command History

Arrow key navigation through 200 previous commands. Persistent via localStorage. Line-numbered history command.

Boot Animation

Simulated BIOS and kernel boot with ASCII art logo. Progressive line reveal with randomized timing. Skippable with any keypress.

GitHub Integration

Repository stats, languages, and profile info fetched daily via GitHub Actions. Hydrated into the virtual filesystem at load.

Fuzzy Suggestions

Levenshtein distance matching for mistyped commands. Did you mean...? suggestions with ranked alternatives.

Permalink Support

URL hash linking to auto-run commands on page load. #neofetch opens straight to system info. Bookmarkable deep links.

Mobile Support

Responsive layout with keyboard toggle button for touch devices. Viewport-aware resizing via ResizeObserver and fit addon.

Platform Capabilities

Works everywhere modern browsers run. Full feature parity across Chrome, Firefox, and Safari. Touch support on mobile.

Architecture

Clean module separation. Each concern in its own file. Zero framework dependencies beyond xterm.js and Vite.

1

Terminal Layer

xterm.js instance with fit addon for viewport sizing, web-links addon for clickable URLs, and Catppuccin Mocha theme applied at initialization.

2

Shell Engine

Input handling with line editing, prompt rendering, and command dispatch. Integrates tab completion, history navigation, and Ctrl shortcuts.

3

Command Registry

Fourteen commands registered with aliases. Levenshtein fuzzy matching for typos. Each command is an independent module returning formatted output.

4

Data Pipeline

GitHub Actions fetch profile and repo data daily via gh CLI. JSON committed to public/data/. Loaded at runtime and hydrated into the virtual filesystem.

Getting Started

Clone, install, and run. No build configuration required. Vite handles everything.

setup.sh 
# Clone and install
$ git clone https://github.com/Real-Fruit-Snacks/Tidepool.git
$ cd Tidepool

# Install dependencies
$ npm install

# Start dev server
$ npm run dev
  VITE v6.0.0  ready in 200ms
  Local: http://localhost:5173/

# Production build
$ npm run build
