FEATURES

A portfolio you can type in

Full xterm.js terminal with virtual filesystem, tab completion, command history, boot animation, and permalink support.

Terminal Emulation

Full xterm.js terminal with VT100/VT220 emulation, 24-bit ANSI color, cursor blink, 1000-line scrollback, and automatic viewport fitting. Clickable URLs via web-links addon.

Virtual Filesystem

Navigate directories, read files, and discover content through a simulated Unix filesystem. Supports ls, cd, cat, and pwd.

GitHub Data Pipeline

Repo stats, languages, and profile info fetched daily via GitHub Actions. Data committed to github.json and hydrated into the virtual filesystem at load time.

Tab Completion

Context-aware completion for commands and file paths. Common prefix expansion with candidate cycling on repeated Tab presses.

ARCHITECTURE

Vite + xterm.js

Modular JavaScript with Vite bundling. Each command is an independent module. GitHub data fetched daily via CI.

1

Terminal Layer

xterm.js with fit and web-links addons. VT100/VT220 emulation, 24-bit color, 1000-line scrollback. Automatic viewport fitting on resize.

2

Shell Engine

Input handling, prompt rendering, command dispatch, and history navigation. Tab completion with context-aware path and command matching.

3

Virtual Filesystem

In-memory directory tree with ls, cd, cat, pwd. Static content plus GitHub data hydrated at load time from github.json.

4

Data Pipeline

GitHub Actions runs daily, fetches repo stats via gh CLI, commits github.json to public/data/. No API tokens exposed to the frontend.

GETTING STARTED

Clone and run

Node.js and npm required. Vite dev server starts in seconds.

build.sh
$ git clone https://github.com/Real-Fruit-Snacks/Tidepool.git
$ cd Tidepool && npm install
$ npm run dev

# Production build
$ npm run build
$ npm run preview
